环境搭建

创建虚拟环境 virtualenv python_env

添加别名:

6 alias virtualenv_python=". /Volumes/MACEXT/python_env/bin/activate && /Volumes/MACEXT/python_env/bin/python"
7 alias virtualenv_pip=". /Volumes/MACEXT/python_env/bin/activate && /Volumes/MACEXT/python_env/bin/pip"

安装包:

virtualenv_pip install django==1.5.1
virtualenv_pip install djangorestframework
virtualenv_pip install markdown
virtualenv_pip install django-filter

如果安装路径不对则修改 /Volumes/MACEXT/python_env/bin/activate

export PYTHONPATH="/Volumes/MACEXT/python_env/site-packages"

Cython简介

cython是一个在python里面混写 c c++ 代码来优化代码执行速度的项目 与其说是一个库, 不如说是一们语言, 其完美地把 python 和 c , c++ 结合在一起, 让你可以用写 python 的方式写 c c++ 我会写一个教程来 讲如何用 cython 优化你的代码

项目参见 cython

数据结构

python代码的速度慢, 是因为python 是动态语言, 在执行过程中需要做很多的类型 判断 比如简单的:

a = 'hello'
b = 'test'
c = a + b

在 a + b 的时候需要先对 a 和 b 的类型进行判断, 如果 a b 为整数则 整数想加, a b 为字符串则 字符串相连, 所以官方推荐的字符串相连的写法是 .join , 如果之前已知 a b 的类型就能减少类型判断这一步,

python 内置的 list dict string set 等数据类型, 给我们编写代码提供很 大便利, 但是这些数据类型为了提供通用性, 有些效率并不高, 比如 dict 的的key 查找基于 __hash__ 函数利用
